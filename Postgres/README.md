## Project: Data Modeling with Postgres

### Project Description

`create_tables.py` Creates a Postgres database with tables designed to optimize queries on song play analysis. 

`etl.py` Created a database schema and ETL pipeline for this analysis. 

`test.ipynb` Created a test notebook to see the final outcome.

### How to run this pipeline
    
    1. Unzip the data.zip file
    
    2. run : python create_tables.py
    
    3. run : python etl.py 
    
    4. open test.ipynb to see the results
